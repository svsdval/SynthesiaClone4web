# SynthesiaClone4web
En:
Another clone for web, but free, with open source, cross-platform and written in Python & JS
![Alt text](img/SynthesiaClone4Web.png?raw=true "main window")
!!This clone created with the help of A.I... Just for fun & testing purposes, but turned out to be fully functional =)
[Check it out https://midishare.svsdval.ru/play](https://midishare.svsdval.ru/play)

So this is the closest version I could make...
Minimal dependencies for backend (mido, python-rtmidi, flask,flask-cors)

However, it does have much of what is needed:

1. Support for multiple MIDI outputs
2. Support for multiple MIDI inputs
3. Support for key illumination (tested on my keyboard)
4. Support for piano sound output using the audio engine, if needed.
4. Loading a MIDI list from the backend
5. Ability to upload your own MIDI files
5. Learning mode
6. Scrolling mode
7. Ability to share a link to a music file
8. Hide/Show unnecessary channels.

Однако в нём есть многое из того что нужно:

4. Загрузка с бэкэнда списка midi
5. Возможность загрузки своих midi
5. Режим обучения
6. Режим прокрутки мелодий
7. Возможность делиться ссылкой на музыку
8. Скрытие / Отображение ненужных каналов.
